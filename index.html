Brendan Eich

Bu inson JavaScript tiliga asos solgan

1961-yil 4-iyull, Pitsburgh, Pensilvaniya, AQSH — amerikalik dasturchi, JavaScript dasturlash tilini yaratuvchisi. 1995-yildan beri Netscape'da ishlagan; Mozilla kompaniyasini tashkil etishda ishtirok etgan, 2014-yilgacha Mozilla korporatsiyasida bosh muhandis bo'lgan.

Math Object

JavaScript’da Math object matematik funksiyalar va konstantalar bilan ishlash uchun mo'ljallangan. U veb ilovalarida ko‘plab matematik amallarni bajarishga yordam beradi. Bu ob'ekt statik bo'lib, Math obyektini bevosita ishlatishingiz mumkin.

Math.PI

Pi qiymatini ifodalaydi, taxminan 3.14159 ga teng.
console.log(Math.PI); // 3.141592653589793

Math.round()

PSonni eng yaqin butunga aylantiradi.
console.log(Math.round(4.7)); // 5
console.log(Math.round(4.4)); // 4

Math.ceil()

Sonni yuqoriga qarab butunga aylantiradi.
console.log(Math.ceil(4.1)); // 5

Math.floor()

Sonni pastga qarab butunga aylantiradi.
console.log(Math.floor(4.9)); // 4

Math.random()

0 va 1 oralig'ida tasodifiy son yaratadi.
console.log(Math.random());

Math.max()

Bir necha argumentlar orasidan eng katta qiymatni qaytaradi.
console.log(Math.max(1, 3, 2)); // 3

Math.min()

Bir necha argumentlar orasidan eng kichik qiymatni qaytaradi.
console.log(Math.min(1, 3, 2)); // 1

Math.pow()

Bir sonni ikkinchisiga daraja qilish uchun ishlatiladi.
console.log(Math.pow(2, 3)); // 8

Math.sqrt()

Sonning kvadrat ildizini qaytaradi.
console.log(Math.sqrt(16)); // 4

Math.abs()

Sonning modulini (musbat qiymat) qaytaradi.
console.log(Math.abs(-5)); // 5

Math.log()

Natural logarifmni (sonning tabiiy logarifmi) qaytaradi.
console.log(Math.log(Math.E)); // 1

Math.exp()

e ni ko'tarilgan darajaga ko'taradi.
console.log(Math.exp(1)); // 2.718281828459045 (e)

Data Types

Primitive
Non-Primitive

String

String: Matnli ma'lumotlarni ifodalaydi. Ikkita qo'shtirnoq yoki bitta qo'shtirnoq ichida yoziladi.
let greeting = "Hello, World!";

Object

Object: Murakkab ma'lumotlarni ifodalaydi va o'z ichiga ma'lumotlar to'plamini oladi.
let person = { name: "John", age: 30, isStudent: true };

Number

Number: Sonli ma'lumotlarni ifodalaydi. Butun sonlar va o'nlik sonlarni o'z ichiga oladi.
let age = 25;
let price = 19.99;

Array

Array: Qiymatlar ro'yxatini ifodalaydi. JavaScript'da bu shuningdek ob'ektning bir turi.
let colors = ["red", "green", "blue"];

Boolean

Boolean: True yoki False qiymatlarini ifodalaydi. Mantiqiy operatsiyalarda qo'llaniladi.
let isValid = true;
let hasPermission = false;

Function

Function: Funksiyalar kodlarni takrorlash yoki qayta ishlash uchun ishlatiladi.
function greet() { console.log("Hello world"); } greet();

Undefined

Undefined: Qiymat aniqlanmagan yoki o'zgartuvchi e'lon qilinganda lekin qiymat belgilanmaganda bo'ladi.
let x;
console.log(x); // undefined

Date

Date: Sana va vaqt bilan ishlash uchun ishlatiladi.
let today = new Date();

Symbol

Symbol: Unikal va o'zgarmas qiymatni ifodalaydi. Ko'pincha identifier sifatida ishlatiladi.
let sym = Symbol('description');

BigInt

BigInt: Juda katta butun sonlarni ifodalash uchun ishlatiladi.
let bigNumber = BigInt(9007199254740991);

Tarixi va rivojlanishi

1995: Brendan Eich JavaScriptni ishlab chiqadi

1996: JavaScript standartlashtiriladi va ECMAScript nomi bilan tanilgan,

2005-yillar: AJAX texnologiyalari bilan JavaScripting ommalashuvi boshlanadi.

2015: ECMAScript 6 (ES6) chiqariladi, bu JavaScripting katta yangilanishdir va yangi sintaksis va funksiyalarni kiritadi.
